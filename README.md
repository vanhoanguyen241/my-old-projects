# My Old Projects

Chào mừng bạn đến với không gian lưu trữ các dự án học tập của mình! Đây là trang web giới thiệu cá nhân được mình xây dựng trong quá trình học tập tại trường.

## Live Demo
Bạn có thể xem trực tiếp tại đây: 
**[https://nguyenvanhoa.eu.org](https://nguyenvanhoa.eu.org)**

## Công nghệ sử dụng
* **HTML5 / CSS3**: Xây dựng cấu trúc và định dạng giao diện.
* **GitHub Pages**: Nền tảng triển khai website (Hosting).
* **Cloudflare**: Quản lý DNS và tối ưu bảo mật HTTPS.

## Chức năng chính
- Giới thiệu thông tin cá nhân.
- Làm trang đích (Landing Page) để giữ chỗ cho tên miền cá nhân.
- Liên kết mạng xã hội và kênh liên lạc.

## Cách chạy dự án
1. **Xem trực tiếp:** Truy cập link demo ở phía trên.
2. **Chạy máy cục bộ:** Tải mã nguồn về máy, nhấp đúp vào file `index.html` để mở trên trình duyệt của bạn.
